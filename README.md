# BMW-Sales-Service-Dashboard-Power-BI:       

The BMW Sales & Service Dashboard is an interactive business intelligence solution developed using Power BI to analyze vehicle sales, service performance, pricing trends, and regional insights. The primary objective of this project was to provide a centralized and user-friendly platform for monitoring key business metrics and supporting data-driven decision-making.

The dataset, sourced from CSV files, includes information on vehicle models, fuel types, sales transactions, service records, and regional dealership performance. Data cleaning and transformation were performed using Power Query, where missing values were handled, data types were corrected, and relevant columns were structured for analysis.

A robust data model was created to establish relationships between different datasets such as sales, service, and region. Using DAX (Data Analysis Expressions), key performance indicators (KPIs) were developed, including total sales, service count, revenue trends, and regional performance metrics.

The dashboard consists of multiple report pages such as Sales Overview, Service Analysis, and Regional Performance. Advanced features like slicers, drill-through functionality, and page navigation using buttons and bookmarks were implemented to enhance user interaction and provide a seamless navigation experience. Visualizations include bar charts, line charts, maps, and KPI cards to represent data clearly and effectively.

This solution enabled the identification of top-performing vehicle models and regions, as well as trends in service demand and pricing. As a result, the dashboard reduced manual reporting efforts by approximately 30% and improved overall sales performance visibility by 25%. It provides valuable insights that help stakeholders make informed business decisions efficiently.
